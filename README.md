# biblioteca-de-estrutura-de-dados
Biblioteca de estruturas de dados implementada 100% do zero em Java (sem java.util.LinkedList/Stack/Queue) — Pilha, Fila, Lista Simples, Lista Dupla, BST, AVL e Árvore B, com aplicação de console para testar e comparar cada estrutura. Projeto acadêmico (UEPB — Estruturas de Dados e Algoritmos).
